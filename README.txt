Weatherford Local Resale & Thrift - Static Demo Site

Files included:
- index.html
- inventory.html
- about.html
- visit.html
- contact.html
- styles.css
- main.js
- assets/images/*

Deployment:
1. Upload the entire folder to Netlify.
2. Set the custom subdomain or connect weatherfordlocalresale.rocksempirellc.com.
3. Update business hours in:
   - index.html
   - visit.html
   - contact.html
4. Optional polish:
   - swap the Facebook page URL for a direct Messenger URL if desired
   - add confirmed phone number if the owner wants it
   - add new inventory photos over time

This demo is built as a clean static site with no framework dependencies.
